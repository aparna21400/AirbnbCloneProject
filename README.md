## ğŸš€ Getting Started

### Prerequisites

- **Programming Language:** JavaScript  
- **Package Manager:** npm  
- **MongoDB Database** (local or Atlas)  
- **Cloudinary Account** (for image uploads)  

---

### Installation

1. Clone the repository:  
git clone https://github.com/aparna21400/AirbnbCloneProject.git    

2. Navigate to the project directory: cd AirbnbCloneProject

Install dependencies: npm install

Create a .env file in the root directory with the following variables: MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_jwt_secret
CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret

Start the server: npm start
